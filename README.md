# PrairieControls
A mod for Stardew Valley that allows you to shoot with the cursor in the arcade minigame "Journey of the Prairie King."

## Installation
 1. [Download the required version of the mod](https://github.com/synthicated/PrairieControls/releases)
 2. Find the game's root directory, where `Stardew Valley.exe` or `StardewValley.exe` is located. You can find the game's root directory [here](https://stardewvalleywiki.com/Modding:Player_Guide/Getting_Started#Find_your_game_folder)
 3. Unzip the mod into the `Mods` folder
 4. Play and have fun - it's easy!

## Technical moments
 - While playing, you may notice some inaccuracies in the bullet direction. Unfortunately, the game code doesn't allow for precise bullet direction without significant modifications. This will take some time;
 - However, your savefile will not break despite using Harmony patches, since only the mini-game process is added
