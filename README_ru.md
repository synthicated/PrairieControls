# PrairieControls
Мод для Stardew Valley который позволяет стрелять курсором в аркадной мини-игре "Путешествия Короля Прерий"

## Установка
 1. [Скачать необходимую версию мода](https://github.com/synthicated/PrairieControls/releases)
 2. Найдите корневой раздел игры, где находится `Stardew Valley.exe` или `StardewValley.exe`.  Как найти корневой раздел игры можно [тут](https://ru.stardewvalleywiki.com/%D0%9C%D0%BE%D0%B4%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D0%B8:%D0%A0%D1%83%D0%BA%D0%BE%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE_%D0%BF%D0%BE_%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8E_%D0%BC%D0%BE%D0%B4%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D0%B9/%D0%92%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5#.D0.9D.D0.B0.D1.85.D0.BE.D0.B6.D0.B4.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BF.D0.B0.D0.BF.D0.BA.D0.B8_.D1.81_.D0.B8.D0.B3.D1.80.D0.BE.D0.B9)
 3. Разархивируй мод в папку `Mods`
 4. Играй и получай удовольствие - всё легко!

## Технические моменты
 - При игре можно заметить неточности в направлении пуль. К сожалению, код игры не позволяет задавать довольно точное направление для пуль без серьёзных изменений. Для этого потребуется некоторое время;
 - Однако, ваше сохранение не сломается не смотря на использование Harmony-патчей, так как лишь добавляется процесс мини-игры
